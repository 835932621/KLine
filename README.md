
# 实现功能
- 蜡烛图，指标图，分时图
- 支持横竖屏
- 界面缩放
- 长按显示十字线，查看指标
- 高仿火币K线的UI实现

# 优点
- 项目实现思路简单，方便二次开发
- 直接使用UIView作为底层容器，所有k线元素都在一个view上绘制，占用内存低
